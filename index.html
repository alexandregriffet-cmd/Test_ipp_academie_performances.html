IPP A4P – Version 3 Premium

Inventaire des Profils de Performance

Cette version premium a été conçue pour des tests terrain sur GitHub avec une interface plus claire, des formulations plus fines et un rapport hybride performance + vie.

Le questionnaire comprend 60 vignettes, un module d’énergie de phase, un module de stress et un contrôle de cohérence par paires miroir discrètes.

Usage conseillé : exploration, accompagnement, entretien de restitution, préparation mentale. Ce test n’est pas encore étalonné sur un échantillon normatif.

Consigne

Pour chaque situation, choisis la réaction qui te ressemble le plus naturellement, même si plusieurs te paraissent acceptables.

Réponds avec sincérité. Le rapport est plus utile quand tu choisis ce qui est spontané, pas ce qui semble idéal.
